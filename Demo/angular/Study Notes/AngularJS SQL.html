跨域 HTTP 请求

如果你需要从不同的服务器（不同域名）上获取数据就需要使用跨域 HTTP 请求。 跨域请求在网页上非常常见。很多网页从不同服务器上载入 CSS, 图片，Js脚本等。 在现代浏览器中，为了数据的安全，所有请求被严格限制在同一域名下，如果需要调用不同站点的数据，需要通过跨域来解决。 以下的 PHP 代码运行使用的网站进行跨域访问。

header("Access-Control-Allow-Origin: *");
